# VITALO Images

## Slideshow Images Needed

Please add the following images to this folder for the homepage slideshow:

1. **slide1.jpg** - Main gym/fitness area (recommended size: 1920x1080px)
2. **slide2.jpg** - Training/workout area (recommended size: 1920x1080px)
3. **slide3.jpg** - Therapy/physiotherapy area (recommended size: 1920x1080px)
4. **slide4.jpg** - Overall facility or additional highlights (recommended size: 1920x1080px)

### Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended dimensions: 1920x1080px (16:9 aspect ratio)
- Maximum file size: 500KB per image (optimize for web)
- Quality: High quality but compressed for fast loading

### How to Add Images:
1. Download your images from Google Drive or other source
2. Rename them to: slide1.jpg, slide2.jpg, slide3.jpg, slide4.jpg
3. Place them in this `/images` folder
4. The slideshow will automatically display them

### Optional: Additional Images
You can add more images for other sections:
- Team member photos
- Facility features
- Equipment
- Classes in action
